Najia 六爻排盘 API 文档

版本: 1.0.0 | 更新时间: 2025年6月20日

概述

Najia 是一个基于 FastAPI 的六爻排盘 Web 服务，提供传统六爻占卜的现代化 API 接口。

项目值
服务地址http://localhost:8000
API 前缀/api
Swagger 文档http://localhost:8000/docs
ReDoc 文档http://localhost:8000/redoc
前端界面http://localhost:8000

API 端点列表

1. 健康检查

GET /api/health

检查服务运行状态。

{ "status": "healthy", "timestamp": "2025-06-20T16:02:00+08:00", "version": "1.0.0" }

2. 手动起卦

POST /api/gua

根据手动输入的参数进行六爻排盘。

请求参数

字段类型必需说明
paramsnumber[]是6个数字数组，每个数字代表一爻
genderstring否性别：男/女/空
titlestring否起卦标题

请求示例

{ "params": [4, 1, 3, 3, 4, 3], "gender": "男", "title": "测试起卦" }

3. 随机起卦

POST /api/random

使用随机数生成六爻排盘。

请求参数

字段类型必需说明
genderstring否性别：男/女/空
titlestring否起卦标题

4. 时间起卦

POST /api/time-gua

根据当前时间进行六爻排盘。

请求参数

字段类型必需说明
genderstring否性别：男/女/空
titlestring否起卦标题

5. 获取常量

GET /api/constants

获取六爻系统中使用的各种常量。

响应示例

{ "success": true, "data": { "tiangan": ["甲", "乙", "丙", "丁", "戊", "己", "庚", "辛", "壬", "癸"], "dizhi": ["子", "丑", "寅", "卯", "辰", "巳", "午", "未", "申", "酉", "戌", "亥"], "wuxing": ["金", "木", "水", "火", "土"], "bagua": ["乾", "坤", "震", "巽", "坎", "离", "艮", "兑"], "liushen": ["青龙", "朱雀", "勾陈", "螣蛇", "白虎", "玄武"], "liuqin": ["父母", "兄弟", "子孙", "妻财", "官鬼"] } }

6. 卦象分析

GET /api/gua-analysis/{gua_name}

根据卦名获取卦象的详细分析信息。

响应数据结构

标准响应格式

{ "success": true, "data": { /* 具体数据 */ }, "rendered": "格式化的排盘结果文本", "message": "操作结果描述" }

主要字段说明

字段说明
name卦名，如"火风鼎"
mark卦象标记，如"011101"
gong所属宫位，如"离"
solar公历时间
lunar农历信息（干支、旬空）
god6六神配置
dong动爻位置
shiy世应位置
qin6六亲关系
bian变卦信息

使用示例

cURL 示例

手动起卦

curl -X POST "http://localhost:8000/api/gua" \ -H "Content-Type: application/json" \ -d '{"params": [4,1,3,3,4,3], "gender": "男", "title": "测试"}'

随机起卦

curl -X POST "http://localhost:8000/api/random" \ -H "Content-Type: application/json" \ -d '{"gender": "女", "title": "随机测试"}'

时间起卦

curl -X POST "http://localhost:8000/api/time-gua" \ -H "Content-Type: application/json" \ -d '{"title": "时间起卦"}'

Python 示例

import requests BASE_URL = "http://localhost:8000/api" # 手动起卦 response = requests.post(f"{BASE_URL}/gua", json={ "params": [4, 1, 3, 3, 4, 3], "gender": "男", "title": "测试起卦" }) result = response.json() print(result["rendered"]) # 获取常量 response = requests.get(f"{BASE_URL}/constants") constants = response.json()["data"] print(constants["bagua"])

JavaScript 示例

async function callAPI() { const response = await fetch('http://localhost:8000/api/gua', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ params: [4, 1, 3, 3, 4, 3], gender: '男', title: '测试起卦' }) }); const result = await response.json(); console.log(result.rendered); }

错误处理

HTTP 状态码

状态码说明
200请求成功
400请求参数错误
404资源不存在
422数据验证失败
500服务器内部错误

错误响应格式

{ "success": false, "detail": "错误详细信息", "message": "操作失败" }

启动服务

开发环境

# 安装依赖 pip install -r requirements.txt # 启动服务（推荐） python start_server.py # 或者直接使用 uvicorn uvicorn app:app --host 0.0.0.0 --port 8000 --reload

Docker 部署

# 构建镜像 docker build -t najia-api . # 运行容器 docker run -d -p 8000:8000 najia-api

测试验证

运行自动化测试脚本验证所有 API：

python test_api_fixed.py

期望输出：所有测试通过 (6/6)

注意事项

  • 参数范围: 手动起卦的 params 数组中每个数字建议在 1-6 之间
  • 编码: 所有文本数据使用 UTF-8 编码
  • 时区: 时间默认使用系统时区（UTC+8）
  • 性能: 单次排盘计算通常在 100ms 内完成
  • 并发: 支持多并发请求，无状态设计
